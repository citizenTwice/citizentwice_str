# C++11 string utils library
Various convenience functions I've (re)written a number of times, for personal use. Mainly std::string and const char* for now, no std::string_view at the moment.

##### Basic
```
 to_upper
 to_lower
 left
 right
 substring_before
 substring_after
 trim
 ltrim
 rtrim
```

##### Easy RegEx matching
```
 re_does_match
 re_get_match
```
 
##### String to/from array
```
 split
 join
```
